## Системные требования

- [Docker](https://docs.docker.com/get-docker/) и Docker Compose
- ОС: Linux, macOS или Windows

## Подготовка и запуск (локальная разработка)
```bash
make setup

# Запуск:
make dev
```

## Тесты и линтер
```bash
make test
make lint
make ci
```
